# PacketManager
Обычная программа на Python которая способна устанавливать программы с помощью ссылок и выбора в cmd. Только для Windows 10-11 x64.


# Соц. сети
> - Viber
> - Telegram
> - Discord
> - Whatsapp (В следующей обнове доделаю, честно!!!)
